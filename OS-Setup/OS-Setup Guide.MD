## Windows

### Install (Windows)

- PowerToys
- Devtoys
- [AutoHotkey](https://www.autohotkey.com/download/ahk-v2.exe) ([AutoHotkey.ahk](Windows%2011/AutoHotkey.ahk))
- [WindHawk](https://ramensoftware.com/downloads/windhawk_setup.exe)

### Change (Windows)

- Install [MacOS cursors by antiden](https://github.com/antiden/macOS-cursors-for-Windows)
- Turn on clipboard history (Win + V)
    (Settings -> Personalisation)
- Turn on transparency effects
- Set accent color to automatic
- Turn on show accent color on title bars and windows borders
- Setup lock screen settings
- Setup window snap (Settings -> System -> Multi-tasking -> SnapWindows -> Snap windows On, all the rest off)
- Setup Start
- Setup WindHawk
    > Start Menu Styler [TranslucentStartMenu](https://github.com/ramensoftware/windows-11-start-menu-styling-guide/tree/main/Themes/TranslucentStartMenu/README.md)
    >
    > Notification Center Styler [TranslucentShell](https://github.com/ramensoftware/windows-11-notification-center-styling-guide/tree/main/Themes/TranslucentShell/README.md)
    >
    >Windows 11 Taskbar Styler[TranslucentTaskbar](https://github.com/ramensoftware/windows-11-taskbar-styling-guide/blob/main/Themes/TranslucentTaskbar/README.md)
    >
    >Taskbar height and icon size
    >
    > ```json
    > TaskbarHeight: 36
    > IconSize: 24
    > TaskbarButtonWidth: 40
    > IconSizeSmall: 10
    > TaskbarButtonWidthSmall: 10
    > ```

- Setup AutoHotKey
    > Win + R
    >
    > `shell:startup`
    >
    > Enter
    >
    > Right click -> New -> Shortcut
    >
    > Browse to `Windows 11\AutoHotkey.ahk` script file.
